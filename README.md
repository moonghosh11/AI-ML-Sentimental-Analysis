# Sentiment Analysis (Python)
## Overview
This project is a simple sentiment analysis program made using Python.  
It takes a sentence from the user and checks whether the sentiment is ‘Positive’, ‘Negative’, or ‘Neutral’.  
The program uses a small list of positive and negative keywords and compares them with the user’s input.

This is a beginner-level AIML project, easy to understand and explain.

---

##  How It Works
1. The user enters a sentence.
2. The program converts the sentence to lowercase.
3. It checks how many positive words are present.
4. It checks how many negative words are present.
5. Based on the counts, it prints one of the following:
   - Positive
   - Negative
   - Neutral
